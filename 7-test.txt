1.arguments是函数的一个属性，只能用在函数内，用来获取函数所有的实参值。对于js来讲无论函数函数有几个参数，只要函数名称一样，都可以看做是同一个函数，而在其他很多编程语言中，函数参数的个数不一样就是不同的函数，所以在函数调用的时候会根据参数个数选择调用对应的函数,对于js来讲可以使用arguments来解决函数参数的个数不一样就是不同的函数问题。

2.call()，apply()，bind()三种方法,它们第一个参数都是指定函数内部中this的指向函数作用域,然后根据指定的作用域，调用该函数,都可以在函数调用时传递参数,在函数调用时call，apply方法是在调用之后立即执行函数，而bind方法没有立即执行
 
3.首先继承得有一个父对象,然后父对象夫指出一个新对象.新对象里面已经有父对象内容,新对象新增内容,扩展新对象子元素继承父元素所有内容就叫做继承.

4.原型(Prototype)它是每一个函数特有的一个属性，该属性其实是一个对象。即每一个函数都有一个原型对象。原型的值可以是一个对象,也可以是null.如果它的值是一个对象,则这个对象也一定有自己的原型.这样就形成了一条线性的链,我们称之为原型链.方法重写就是在同一个类中,方法的名字相同,但参数个数、参数的类型或返回值类型不同,这计生方法重写.

5.基本类型变量在引用的时候引用的是变量的值(相当于把变量copy一份)，所以当引用的变量发生更改时不会影响引用者.引用类型(对象类型)相互引用时，引用的是对象的内存地址，所以当被引用的对象发生变化时，会影响到引用者。

6.art-template是实用的js模板引擎,我么你可以套用它的模式进行页面美化,同时使我们的页面富有多彩,灵活.

7.闭包是一个拥有许多变量和绑定了这些变量的环境的表达式（通常是一个函数），因而这些变量也是该表达式的一部分。闭包的作用就是在函数执行完并返回后，闭包使得Javascript的垃圾回收机制不会收回函数所占用的资源,这使得页面内存负压大大减少.

8.typeof是一种运算符，返回值为字符串,该字符串用来说明运算数的数据类型,instanceof用于判断某个变量是否是某个对象的实例，返回值为true或false.

9.http协议即超文本传输协议（HTTP，HyperText Transfer Protocol)是互联网上应用最为广泛的一种网络协议。所有的WWW文件都必须遵守这个标准。HTTP是一个客户端和服务器端请求和应答的标准（TCP）.

10.get请求数据会附在URL之后(就是把数据放置在HTTP协议头中)，以?分割URL和传输数据，多个参数用&连接,而post请求把提交的数据放置在是HTTP包的包体中

11.Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境。Node.js 使用了一个事件驱动、非阻塞式 I/O 的模型,使其轻量又高效.npm则是随从node.js
一同安装的包管理器.

12.npm install express ―save的作用是在服务段安装一个express模块.npm uninstall express用于删除express模块.

13.js中代码执行不按顺序，‘跳过’执行，待其他某些代码执行完后，再来执行，称为“异步”。同步至代码一次执行.同步操作会一致等待上个命令结束才执行下个命令会使我们页面卡顿,异步操作不会那样页面会很灵活不会出现卡顿等情况.

14.action和mehod,action是网页提交数据所连接的地址,method指的获取数据的方式有get和post请求.

15.AJAX即“Asynchronous Javascript And XML”（异步JavaScript和XML),是指一种创建交互式网页应用的网页开发技术。ajax页面可以传送json格式的数据，服务端可以响应json对象，json数组,然后通过这些数据,数组什么的来和服务端进行数据交互.

16. 静态页面是网页的代码都在页面中，不需要执行asp,php,jsp,.net等程序生成客户端网页代码的网页即我们前端所写的网页,动态页面是通过服务端执行asp,php,jsp,.net等程序生成客户端网页代码的网页.

17.JSON（JavaScript Object Notation）是一种轻量级的数据交换格式。它基于ECMAScript的一个子集。 JSON采用完全独立于语言的文本格式。JS 中 JSON 的两个转化方法分别是JSON.stringify和JSON.parse.
